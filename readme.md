## 📺 Descrição Geral do Sistema
O sistema consiste em uma plataforma de streaming de vídeo que divide seus usuários entre os planos Padrão e Premium. A interface permite que administradores façam o gerenciamento completo do catálogo (Filmes, Séries e Animações), controlando quais conteúdos são produções originais ou licenciadas, além de gerenciar restrições de lançamentos e qualidades gráficas baseadas no plano do cliente.
